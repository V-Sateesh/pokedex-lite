# Pokédex Lite

Simple React + TypeScript + Vite web app that lists Pokémon from the PokéAPI with:

- Pagination
- Search by name
- Filter by type
- Favorites (persisted in localStorage)
- Detail modal with stats and abilities
- Responsive layout

## Requirements

- Node.js 18+
- npm or pnpm or yarn

## Setup

```bash
npm install
npm run dev
```

Then open the URL printed in your terminal (usually http://localhost:5173).

## Build for production

```bash
npm run build
npm run preview
```
